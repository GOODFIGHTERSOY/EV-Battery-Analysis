# Smart EV Battery Analysis System

This project analyzes battery performance for low-cost electric vehicles.

## Features
- Power and energy calculation
- Battery discharge analysis
- Range estimation
- Data visualization using graphs
- Interactive UI using Streamlit

## Technologies Used
- Python
- Pandas
- Matplotlib
- Streamlit

## How to Run
```bash
python -m pip install pandas matplotlib streamlit
python battery_analysis.py
python -m streamlit run app.py
